第三章 复变函数积分

  • 定义转化为线积分：∫Cf(z)dz=∫C(udx−vdy)+i∫C(vdx+udy)\int_C f(z)dz = \int_C (u dx - v dy) + i \int_C (v dx + u dy)∫C​f(z)dz=∫C​(udx−vdy)+i∫C​(vdx+udy)

2. 柯西定理 (Cauchy-Goursat)

  • 定理： 设 f(z)f(z)f(z) 在单连通区域 DDD 内解析，CCC 为 DDD 内任一封闭曲线，则 ∮Cf(z)dz=0\oint_C f(z)dz = 0∮C​f(z)dz=0。
  • 强化形式： 在区域 DDD 解析，在闭包 Dˉ\bar{D}Dˉ 上连续，则沿边界积分 ∮∂Df(z)dz=0\oint_{\partial D} f(z)dz = 0∮∂D​f(z)dz=0。

3. 复合闭路定理与变形定理

  • 连续变形定理： 解析函数的积分路径可以在不穿过奇点（非解析点）的前提下任意连续变形，积分值不变。
  • 复合闭路定理（多连通区域）： 外边界逆时针，所有内边界顺时针积分之和为0；或外边界积分等于所有包围奇点的内部小闭曲线（同向）积分之和。

4. 原函数与不定积分

  • 定理： 在单连通区域内解析的函数必有原函数，存在积分路径无关性。
  • 牛顿-莱布尼茨公式 (N-L公式) & 分部积分： 完全适用，前提是路径所在的区域必须是单连通且函数在其中解析。

💡 做题方法与技巧

  • 复变积分首要思考步骤：
    1. 判别被积函数有无奇点。
    2. 奇点是否在积分回路内部？（在外 →\to→ 解析 →\to→ 积分为0；在内 →\to→ 圈起来 →\to→ 用公式）。
  • 初等函数（如 znz^nzn）可以使用 N-L 公式，甚至可以使用多项式展开，前提是它存在原函数形式。

第五章 留数 (Residue)

1. 孤立奇点分类

设 z0z_0z0​ 是 f(z)f(z)f(z) 的孤立奇点，考察其在去心邻域的 Laurent 展开式：

  • I. 可去奇点： 不含负幂项。
    •   ⟺  lim⁡z→z0f(z)\iff \lim_{z \to z_0} f(z)⟺limz→z0​​f(z) 存在且有限。
    •   ⟺  f(z)\iff f(z)⟺f(z) 在邻域内有界。
  • II. 极点 (Pole)： 含有有限个负幂项，最高负幂次为 −m-m−m（称 mmm 阶极点）。
    •   ⟺  f(z)=ϕ(z)(z−z0)m\iff f(z) = \frac{\phi(z)}{(z-z_0)^m}⟺f(z)=(z−z0​)mϕ(z)​（ϕ(z0)≠0\phi(z_0)\neq 0ϕ(z0​)=0 解析）。
    •   ⟺  lim⁡z→z0f(z)=∞\iff \lim_{z \to z_0} f(z) = \infty⟺limz→z0​​f(z)=∞。
    •   ⟺  z0\iff z_0⟺z0​ 是 1/f(z)1/f(z)1/f(z) 的 mmm 阶零点。
  • III. 本性奇点： 含有无穷多个负幂项。

2. 留数及其计算

  • 定义： f(z)f(z)f(z) 在 z0z_0z0​ 处的留数定义为 Laurent 展开式中 (z−z0)−1(z-z_0)^{-1}(z−z0​)−1 的系数 C−1C_{-1}C−1​，记作 Res[f(z),z0]\text{Res}[f(z), z_0]Res[f(z),z0​]。
  • 留数定理： ∮Cf(z)dz=2πi∑k=1nRes[f(z),zk]\oint_C f(z)dz = 2\pi i \sum_{k=1}^n \text{Res}[f(z), z_k]∮C​f(z)dz=2πi∑k=1n​Res[f(z),zk​]

留数计算规则 (Rules)：

  • Rule I（简单极点）： Res[f(z),z0]=lim⁡z→z0(z−z0)f(z)\text{Res}[f(z), z_0] = \lim_{z \to z_0} (z-z_0) f(z)Res[f(z),z0​]=limz→z0​​(z−z0​)f(z)
  • Rule II（简单极点分式型）： 设 f(z)=P(z)Q(z)f(z) = \frac{P(z)}{Q(z)}f(z)=Q(z)P(z)​，P(z0)≠0,Q(z0)=0,Q′(z0)≠0P(z_0)\neq 0, Q(z_0)=0, Q'(z_0)\neq 0P(z0​)=0,Q(z0​)=0,Q′(z0​)=0，则 Res[f(z),z0]=P(z0)Q′(z0)\text{Res}[f(z), z_0] = \frac{P(z_0)}{Q'(z_0)}Res[f(z),z0​]=Q′(z0​)P(z0​)​。
  • Rule III（mmm 阶极点）： Res[f(z),z0]=1(m−1)!lim⁡z→z0dm−1dzm−1[(z−z0)mf(z)]\text{Res}[f(z), z_0] = \frac{1}{(m-1)!} \lim_{z \to z_0} \frac{d^{m-1}}{dz^{m-1}} [ (z-z_0)^m f(z) ]Res[f(z),z0​]=(m−1)!1​limz→z0​​dzm−1dm−1​[(z−z0​)mf(z)]

💡 做题方法与技巧

  • 求留数策略选择： 如果是 1 阶极点，无脑用 Rule I 或 Rule II。 如果是高阶极点（m≥3m \ge 3m≥3），强烈不建议用 Rule III（笔记批注：“导多不用”），因为高阶求导极易出错。推荐方法：直接做局部的 Taylor/Laurent 级数展开，寻找 1z−z0\frac{1}{z-z_0}z−z0​1​ 项的系数 C−1C_{-1}C−1​，速度快且准确。
  • 遇到本性奇点： 只能通过展开 Laurent 级数找 C−1C_{-1}C−1​，没有任何捷径公式。
